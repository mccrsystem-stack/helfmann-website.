# Grok SaaS Starter

Produktionsnahes Starter-Projekt fuer eine mehrmandantenfaehige Grok-Chat-App mit Next.js 15, Clerk, Supabase und Stripe-Vorbereitung.

## Geplanter Ausbau

- Auth mit Clerk
- Dashboard fuer eigene Chats
- Streaming Chat via xAI Grok API
- Persistenz in Supabase
- Rate Limiting und Billing-Grundlagen
